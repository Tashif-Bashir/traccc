Get current C\+U\+DA device number.

This function wraps the cuda\+Get\+Device function in a way that returns the device number rather than use a reference argument to write to.

Note that calling the function on a machine with no C\+U\+DA device does not result in an error, the function just returns -\/1 in that case.
