\subsubsection*{template$<$typename R, typename... A$>$\newline
class traccc\+::algorithm$<$ R(\+A...)$>$}

Unified algorithm semantics which convert an input to an output. 

This class provides a single, unified semantic for algorithms that can be used throughout the traccc code. This virtual class is templated with an input type and an output type, and any class implementing it is expected to be able to transform the input type into the output type in one way or another.
