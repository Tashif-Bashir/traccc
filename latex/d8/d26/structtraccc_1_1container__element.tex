\subsubsection*{template$<$typename header\+\_\+reference\+\_\+t, typename vector\+\_\+reference\+\_\+t$>$\newline
struct traccc\+::container\+\_\+element$<$ header\+\_\+reference\+\_\+t, vector\+\_\+reference\+\_\+t $>$}

View class for an element in a header-\/vector container. 

In order to enforce certain invariants on the {\ttfamily container} header-\/vector type, we access elements (which, of course, are product types of a header and a vector) through this wrapper class. This class provides low-\/overhead access to the struct-\/of-\/arrays container class to emulate an array-\/of-\/structs architecture.
