\doxysubsection{Detailed Description}
Connected component labelling

Note that the separation between the public and private interface is only there in the class because the compilers can\textquotesingle{}t automatically figure out the \char`\"{}vector type\char`\"{} of the templated implementation, without adding a lot of \char`\"{}internal knowledge\char`\"{} about the vector types into this piece of code. So instead the public operators are specifically implemented for the host-\/ and device versions of the E\+DM, making use of a single implementation internally.
