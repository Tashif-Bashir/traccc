\doxysubsection{Detailed Description}
Wrapper class for {\ttfamily sycl\+::queue} 

It is necessary for passing around S\+Y\+CL queue objects in code that should not be directly exposed to the S\+Y\+CL headers.

Note that unlike {\ttfamily vecmem\+::sycl\+::queue\+\_\+wrapper}, this type can not own a queue of its own. It can only view a queue that is owned by \char`\"{}somebody else\char`\"{}.
