Algorithm to copy a container from a device to the host

Specialisations of this algorithm can be used to get a container back from a device to the host. Ideally at the end of an algorithm chain that would\textquotesingle{}ve produced something in device memory.

Note that the algorithm needs to treat constant views and buffers a little differently, so the user should make sure to pass in the correct type.
