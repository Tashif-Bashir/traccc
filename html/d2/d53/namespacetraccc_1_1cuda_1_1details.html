int traccc::cuda::details::get_device ( )

Get current CUDA device number.

This function wraps the cudaGetDevice function in a way that returns the device number rather than use a reference argument to write to.

Note that calling the function on a machine with no CUDA device does not result in an error, the function just returns -1 in that case.
