Detailed Description

template<typename detector_t>
class traccc::sycl::experimental::spacepoint_formation< detector_t >

Algorithm forming space points out of measurements

This algorithm performs the local-to-global transformation of the 2D measurements made on every detector module, into 3D spacepoint coordinates.
