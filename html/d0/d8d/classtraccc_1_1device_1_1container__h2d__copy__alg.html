Detailed Description

template<typename CONTAINER_TYPES>
class traccc::device::container_h2d_copy_alg< CONTAINER_TYPES >

Algorithm to copy a container from the host to the device

Specialisations of this algorithm can be used to copy a container to a device for a subsequent algorithm that expects its input to already be on the device that it wants to run on.

Note that the input type is not a "host type", but rather a constant view. This is meant to allow using this algorithm on top of more complicated data objects.
