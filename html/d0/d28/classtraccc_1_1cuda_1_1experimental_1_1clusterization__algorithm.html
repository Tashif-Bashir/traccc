Detailed Description

Algorithm performing hit clusterization

This algorithm implements hit clusterization in a massively-parallel approach. Each thread handles a pre-determined number of detector cells.

This algorithm returns a buffer which is not necessarily filled yet. A synchronisation statement is required before destroying this buffer.
