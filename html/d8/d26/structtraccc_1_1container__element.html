Detailed Description

template<typename header_reference_t, typename vector_reference_t>
struct traccc::container_element< header_reference_t, vector_reference_t >

View class for an element in a header-vector container.

In order to enforce certain invariants on the container header-vector type, we access elements (which, of course, are product types of a header and a vector) through this wrapper class. This class provides low-overhead access to the struct-of-arrays container class to emulate an array-of-structs architecture.
