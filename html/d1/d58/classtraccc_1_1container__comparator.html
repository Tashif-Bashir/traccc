Detailed Description

template<typename HEADER_TYPE, typename ITEM_TYPE>
class traccc::container_comparator< HEADER_TYPE, ITEM_TYPE >

Functor comparing two containers, and printing the results to the output

This is meant to be used in the example applications for nicely comparing the results made on the host and on a device. Though the code actually allows comparisons between any two containers.

Template Parameters
HEADER_TYPEThe header type in the container
ITEM_TYPEThe item type in the container
