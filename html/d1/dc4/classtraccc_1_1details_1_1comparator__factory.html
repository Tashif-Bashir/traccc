Detailed Description

template<typename TYPE>
class traccc::details::comparator_factory< TYPE >

Factory creating instances of "comparator objects" for a given type

This level of abstraction is necessary to be able to construct comparator objects that would have extra configuration parameters over the reference object and the comparison uncertainty.

Template Parameters
TYPEThe type for which a comparator object should be generated
