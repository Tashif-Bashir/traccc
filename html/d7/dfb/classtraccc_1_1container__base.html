Detailed Description

template<typename header_t, typename item_t, template< typename > class vector_t, template< typename > class jagged_vector_t, template< typename, typename > class pair_t = thrust::pair>
class traccc::container_base< header_t, item_t, vector_t, jagged_vector_t, pair_t >

Container base class for describing objects in a given event

This is the generic container of the code, holding all relevant information about objcts in a given event.

It can be instantiated with different vector types, to be able to use the same container type in both host and device code.

It also can be instantiated with different edm types represented by header and item type.
