Detailed Description

template<typename header_t, typename item_t>
struct traccc::container_view< header_t, item_t >

Structure used to send the data about the container to device code

This is the type that can be passed to device code as-is. But since in host code one needs to manage the data describing a traccc::container either using traccc::container_data or traccc::container_buffer, it needs to have constructors from both of those types.

In fact it needs to be created from one of those types, as such an object can only function if an instance of one of those types exists alongside it as well.
