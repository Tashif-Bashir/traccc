Detailed Description

template<typename T>
class traccc::details::is_same_object< T >

Generic unary functor for checking equality with a certain object

The idea is to use this class as-is for types that have a well-defined equality operator, and to specialise it for cases where the uncertainty of the equality can be set by the user. (To account for floating point uncertainties in the algorithms.)
